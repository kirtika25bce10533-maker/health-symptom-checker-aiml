Health Symptom Checker (AI/ML Project)

Overview:
This project predicts diseases like Flu, Viral Fever, Infection, Cold, and Allergy using Machine Learning.

Objectives:
- Build a disease prediction system
- Include viral fever and infection
- Provide quick predictions

Features:
- Symptom-based input (1 = Yes, 0 = No)
- Predicts disease
- Includes accuracy
- Easy to use

Tech Stack:
- Python
- Pandas
- Scikit-learn

Machine Learning Model:
- Decision Tree Classifier
- Supervised Learning

Symptoms Used:
Fever, Cough, Headache, Fatigue, Body Pain, Sore Throat, Runny Nose

Diseases Covered:
Flu, Viral Fever, Infection, Cold, Allergy

How It Works:
1. Train model on dataset
2. User enters symptoms
3. Model predicts disease

How to Run:
1. Install: pip install pandas scikit-learn
2. Run: python app.py
3. Enter symptoms

Advantages:
- Simple and fast
- Useful for awareness

Limitations:
- Small dataset
- Not fully accurate
- Not a replacement for doctors

Future Improvements:
- Bigger dataset
- GUI or web app
