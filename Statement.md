Problem Statement

Overview
This project aims to develop a machine learning-based system that predicts possible diseases based on user-input symptoms. The system is designed to assist users in identifying potential health conditions such as Flu, Viral Fever, Infection, Cold, and Allergy.

Problem Description
In daily life, people often ignore early symptoms like fever, cough, or fatigue due to lack of awareness or time. This delay can lead to worsening of health conditions. Additionally, not everyone has immediate access to medical professionals for basic consultation.
There is a need for a simple, accessible, and intelligent system that can analyze symptoms and provide an initial prediction of possible diseases.

Objective
.To build a machine learning model that predicts diseases based on symptoms
.To include common illnesses such as Flu, Viral Fever, Infection, Cold, and Allergy
.To provide quick and easy predictions for users
.To demonstrate the practical application of AI in healthcare

Input and Output

Input:
User provides symptoms as input (1 = Yes, 0 = No):
.Fever
.Cough
.Headache
.Fatigue
.Body Pain
.Sore Throat
.Runny Nose

Output:
.Predicted disease (Flu, Viral Fever, Infection, Cold, Allergy)

Approach
The project uses a Decision Tree Classifier to analyze the relationship between symptoms and diseases. The model is trained on a dataset containing different combinations of symptoms and their corresponding diseases.
Steps:
1. Create dataset
2. Train ML model
3. Evaluate accuracy
4. Take user input
5. Predict disease
Real-World Relevance
This project addresses a real-world problem by providing a basic health assistance tool. It can be useful for:
.Students and beginners learning AI/ML
.Individuals seeking quick health insights
.Awareness about early disease detection
