Advanced Health Symptom Checker (AI/ML Project)

• Overview:
The Advanced Health Symptom Checker is a Machine Learning-based project that predicts possible diseases based on user-input symptoms.
It helps in providing basic health awareness by identifying common illnesses such as:
- Flu  
- Viral Fever  
- Infection  
- Cold  
- Allergy  

Note: This project is for educational purposes only and does not replace professional medical advice.

• Objectives:
- Build a disease prediction system using Machine Learning  
- Include common diseases like viral fever and infections  
- Provide quick and simple predictions  
- Demonstrate AI application in healthcare  


• Features:
- Takes symptom inputs (1 = Yes, 0 = No)  
- Predicts the most probable disease  
- Includes viral fever and infection detection  
- Displays model accuracy  
- Beginner-friendly  


• Tech Stack:
- Python  
- Pandas  
- Scikit-learn  



• Machine Learning Model:
- Algorithm: Decision Tree Classifier  
- Type: Supervised Learning  
- Task: Classification  



• Dataset Information:

Symptoms:-
- Fever  
- Cough  
- Headache  
- Fatigue  
- Body Pain  
- Sore Throat  
- Runny Nose  

Diseases:-
- Flu  
- Viral Fever  
- Infection  
- Cold  
- Allergy  

Each symptom is represented as:-
- 1 → Yes  
- 0 → No  



• How It Works:
1. Model is trained on dataset  
2. User enters symptoms  
3. Model processes input  
4. System predicts disease  



• How to Run?
~Step 1: Install dependencies
pip install pandas scikit-learn

~Step 2: Run the program
python app.py

~Step 3: Enter symptoms (1 or 0)



Example:

Input:
Fever: 1  
Cough: 1  
Headache: 1  
Fatigue: 1  
Body Pain: 1  
Sore Throat: 1  
Runny Nose: 0  

Output:
Predicted Disease: Viral Fever
